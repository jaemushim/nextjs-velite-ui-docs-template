In this repo is a simple implementation of a markdown static docs, a demo can be seen [here](https://my-first-blog-dusky.vercel.app/), Features include:
- MDX Components
- Shadcn UI
- Search (fuse.js)
- table of contents
- Syntax Highlighting in code block
